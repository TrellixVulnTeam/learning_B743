| `()` | round / open brackets |

## text editor

1. [你一定要知道的几款经典文本编辑器](https://mp.weixin.qq.com/s?__biz=MzI1NTI4OTIxMA==&mid=2247484153&idx=1&sn=9a7cc3365b65ee95a39e1442b028a59c&scene=21#wechat_redirect)
2. MATLAB 编辑器
   * 语法高亮
   * 行号显示
   * 支持断点、单步调试
   * 编辑脚本、函数、类、枚举等MATLAB文件类型
   * 拖拉编辑
   * 批量注释
3. UltraEdit
   * 界面友好
   * 支持语法高亮
   * 支持代码折叠
   * 支持宏
   * 列模式
4. notepad++
   * 自动补全
   * 语法高亮
   * 拖放特性
   * 搜索
   * 缩放
5. Vim
   * 代码拼写检查
   * 语法高亮
   * 显示行编号
   * 改变界面颜色
6. Emacs
   * 命令来管理语句和段落
   * 语法高亮
   * 键盘宏
7. TextMate-MacOS
   * 代码自动完成
   * 可直接在文档中运行SHELL 命令
   * 支持多种风格
   * 支持宏
8. 其他
