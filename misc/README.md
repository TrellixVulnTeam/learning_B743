# misc

该目录存放的内容具有如下特点

1. 叶子节点软件，即不包含基于该软件的库
2. 仅提供命令行，而没有创造一门新的编程语言（这个界定非常不清晰）
3. 取消操作系统的划分，即合并之前的`linux/`目录

4. text editor
   * `TextMate`
   * `Emacs`
   * `Vim`
   * `notepad++`
   * `UltraEdit`
   * PSPad
   * TotalEdit
   * NoteTab
   * Gedit
   * Komodo
   * Nvu
   * TextPad
   * EditPlus
   * BBEdit
   * GEANY
   * ECLIPSE
   * JEDIT
